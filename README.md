# ChatGPT

GUI for ChatGPT API https://huggingface.co/spaces/AutoGeneralAI/ChatGPT

## 使用方法
将自己的OpenAI API KEY https://platform.openai.com/
放入key输入框，然后就可以愉快的使用ChatGPT，愉快的对话了。

本项目不是web版的ChatGPT，而是调用其官方API的GUI for ChatGPT API。
